# Scancode Set 1 Reference (IBM PC / PS/2)

- **Make Code (Press):** byte sent when key is pressed  
- **Break Code (Release):** `Make Code + 0x80`  
- **Extended Keys:** prefixed with `0xE0`  
- **Used in:** PS/2 keyboards, legacy BIOS, bare-metal OS projects

## Notes

- PrintScreen / Pause are multi-byte sequences (not included)
- Keyboard buffer is only 1 byte → must be read in IRQ handler
- Set 1 is legacy PS/2 / BIOS compatibility mode
- Some laptops may send extended sequences (E0/E1)
